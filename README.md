# Algebra Explorer

Algebra Explorer is a beginner-friendly interactive web app that helps students learn algebra step by step.

## What students can learn

- Combine like terms in expressions
- Solve linear equations in the form `ax + b = cx + d`
- Apply the distributive property in `k(mx + n)`
- Practice with randomly generated equation challenges

## How to run

1. Open `index.html` in any modern browser.
2. Use the topic tabs to switch between lessons.
3. Enter values and click **Show Steps**.
4. Try the quick challenge section for practice.

## Teaching approach

- Each topic explains every move in simple language.
- Students see both the process and the final answer.
- Practice mode reinforces confidence through repetition.

## Files

- `index.html` - App structure and lesson sections
- `styles.css` - Visual design and responsive layout
- `app.js` - Algebra logic, step generation, and challenge mode
